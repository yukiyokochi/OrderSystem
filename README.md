### 概要
都内飲食店様より受注したオンライン料理注文システム開発案件の納品物を、ポートフォリオ用Webアプリケーションとしてリメイクしました。
### 機能一覧
- 料理メニューの表示・詳細表示・追加・削除・編集
- 料理メニューの売り切れ情報の反映
- 料理メニューのカテゴリー機能
- 登録住所別の配達価格の反映
- カート機能
- フォーム機能
- クレジットカード決済機能
- 従業員用顧客情報表示機能
### 使用技術一覧
- 開発言語,フレームワーク: Python3, Django2, HTML5, Sass, JavaScript
- インフラ: AWS EC2
- データベース: PostgreSQL
- APサーバ: Gunicorn
- Webサーバ: Nginx
